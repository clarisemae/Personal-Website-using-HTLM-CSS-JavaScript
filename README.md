## Features

* **Interactive Instagram Frame:** A custom-styled profile section that mimics an Instagram post layout with an image carousel.
* **Falling Cat Animation:** A fun, JavaScript-powered background animation featuring falling cats üê±.
* **Skills Showcase:** clearly organized sections for Soft Skills, Technical Skills, and Tools using modern card layouts.
* **Responsive Design:** Adjusts layout for smaller screens.

## Technologies Used

* **HTML5** - For structure and semantic markup.
* **CSS3** - For styling, including Flexbox for layout and custom animations.
* **JavaScript** - For the interactive image carousel and the falling cat background effects.
* **FontAwesome** - For icons (social media, heart, comment, etc.).
